To run the automated project startup script:

1. Open a PowerShell terminal in the project root.
2. Run the following command:
```powershell
./run.bat
```

// turbo
3. Alternatively, run the project directly through here:
```powershell
./run.bat
```

This will automatically:
- Close any stale backend/frontend processes.
- Launch the FastAPI Backend on [http://localhost:8000](http://localhost:8000).
- Launch the React Frontend on [http://localhost:5178](http://localhost:5178).
